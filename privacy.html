Privacy Policy for Juizo

Last Updated: January 31, 2026

1. Introduction

Juizo ("we", "our", or "the add-in") is a Microsoft Excel add-in developed by Chris Aragao. This Privacy Policy explains how we handle information when you use Juizo.

2. Information We Collect

We do not collect, store, or transmit any personal information or data.

Juizo operates entirely within your local Excel environment:

  • All data processing happens locally on your device
  • Your Excel data never leaves your computer
  • No analytics, tracking, or telemetry is collected
  • No user accounts or authentication required
  • No data is sent to external servers

3. How Juizo Works

Juizo is a client-side application that:

  • Loads as a task pane within Excel
  • Processes your Excel table data using JavaScript libraries (Arquero, Vega-Lite, Mermaid)
  • Displays visualizations and transformations within Excel
  • Stores query and chart specifications locally in Excel document settings

4. Local Storage

Juizo uses Excel's document settings API to save:

  • Your Arquero queries (associated with specific tables)
  • Your Vega-Lite chart specifications
  • Your Mermaid diagram definitions

This data is stored within your Excel workbook file and remains under your control.

5. Code Execution

Important Security Notice: The Arquero Query Editor allows you to write and execute JavaScript code to transform your data. This code:

  • Executes locally in your browser environment
  • Has access only to the Excel table data you select
  • Is executed by you, for your own use
  • Is not transmitted, stored, or reviewed by us

You are responsible for the code you write and execute. Only run code from trusted sources.

6. Third-Party Libraries

Juizo includes the following open-source JavaScript libraries that run client-side:

  • Arquero - Data transformation (BSD 3-Clause License)
  • Vega-Lite - Data visualization (BSD 3-Clause License)
  • Mermaid - Diagram generation (MIT License)

These libraries are bundled with the add-in and do not collect or transmit data.

7. Microsoft Office Integration

Juizo uses the Microsoft Office JavaScript API to:

  • Read data from Excel tables
  • Write transformed data back to Excel
  • Insert charts and diagrams as images
  • Save settings within Excel documents

These operations are subject to Microsoft's Excel and Office 365 privacy policies.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted at https://cbaragao.github.io/Juizo/privacy.html with an updated "Last Updated" date.

9. Contact

If you have questions or concerns about this Privacy Policy, please contact us via:

  • GitHub Issues: https://github.com/cbaragao/Juizo/issues

10. Your Rights

Since Juizo does not collect any personal data, there is no data for us to access, modify, or delete. You have complete control over:

  • Your Excel data (managed through Excel)
  • Saved queries and specifications (stored in your Excel workbook)
  • The add-in itself (can be uninstalled at any time)
